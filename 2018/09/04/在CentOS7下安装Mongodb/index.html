在CentOS7下安装Mongodb

9月 4, 2018

最近刚开始学习CentOS的部署和Mongodb,特地记录一下

进入Mongodb官网 点击右上角 Get MonogDB

选择 Community Server

选择Linux 下载Linux64-bit legacy x64

将下载的文件放入centos7中后解压缩文件

在解压后文件夹新建数据文件夹data和log文件夹

bin目录下新建配置文件

注意dbpath和logpath是你解压后文件夹位置

开启端口

查看端口

重启防火墙

最后进入解压缩文件夹中bin目录下即可启动成功

随后制作开机启动

在/lib/systemd/system/目录下新建mongodb.service文件

注意ExecStart和ExecStop中路径是你解压后文件夹位置

设置权限

启动关闭服务，设置开机启动

大功告成！这样就可以简单的部署MongoDB并随着CentOS7启动而自动开启了！
